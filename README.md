# Getting Started

Navigate to the app root directory:

```bash
cd fetch-users
```

IMPORTANT: Make sure you are running node version 12.22.0 or above.

Get the necessary dependencies:

```bash
npm install
# or 
yarn install
```

## Run The App

```bash
npm run dev
# or 
yarn dev
```

Open http://localhost:3000 in your browser to use the app.

## Run App Tests

```bash
npm run test
# or 
yarn test
```
